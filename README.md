# Framework for simplifying Classically Controlled Operations in Dynamic Circuits 

This tool allows to simplify and remove classically controlled operations in dynamic circuits at compile time. This optimization pass simulates a dynamic circuits with an algorithm based on [Quantum Constant Propagation](https://link.springer.com/chapter/10.1007/978-3-031-44245-2_9). The information gathered from the simulation is used to remove classical controlled operations or replace them exploiting the Probabilistic Circuit Model ([1](https://ieeexplore.ieee.org/document/10821051)[2](https://ieeexplore.ieee.org/stamp/stamp.jsp?arnumber=10821341))
